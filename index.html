📖 API Keys Setup Guide
1
📊 Finnhub API Key Required
Go to finnhub.io/register
Sign up with email or Google (free).
After login, your API key is shown on the Dashboard page.
Copy the key and paste it in the app's ⚙️ Settings panel.
Free tier: 60 API calls/minute. Enough for tracking ~15-20 stocks with auto-refresh.
2
🤖 Groq API Key Optional — AI features
Go to console.groq.com/keys
Sign up (free) with Google, GitHub, or email.
Click Create API Key → name it anything → Submit.
Copy the key immediately (it won't be shown again). Paste it in ⚙️ Settings.
Free tier: 14,400 requests/day. Powers AI news analysis, analyst summaries, macro impact, earnings call summaries, and the AI Analyst Verdict.
3
✨ Google Gemini API Key Optional — AI boost
Go to aistudio.google.com/apikey
Sign in with Google and click Create API Key.
Copy the key and paste it in ⚙️ Settings.
Free tier: 15 RPM, 1,500 RPD. Offloads news, analyst, and macro AI to Google — reduces Groq rate limit pressure by ~60%.
4
📈 Alpha Vantage API Key Optional — extra data
Go to alphavantage.co/support/#api-key
Fill in your name, email, and pick "Free" plan. Click Get Free API Key.
Your key appears on screen and is emailed to you. Paste it in ⚙️ Settings.
Free tier: 25 calls/day. Used for company overview, earnings transcripts, technical indicators (RSI/MACD/SMA), and revenue trends. Use sparingly — buttons load on-demand.
